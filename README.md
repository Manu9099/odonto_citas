# Odonto Citas

Sistema web para gestión de citas odontológicas con roles de paciente, odontólogo y administrador.

## Stack
Backend: Java 21, Spring Boot, PostgreSQL, JWT, Flyway.
Frontend: React, TypeScript, Vite, React Query.

## Módulos
- Autenticación
- Gestión de pacientes
- Gestión de odontólogos
- Horarios disponibles
- Reserva de citas
- Calendario diario
- Pagos
- Recordatorios por correo
